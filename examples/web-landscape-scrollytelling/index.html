视觉锚点不依赖卡片堆砌，而是通过一个强场景去压住整页气质。

横版网页的关键，不是“宽”，而是让宽屏看起来像一块有重量的展厅墙面。

这里的 section 不走常规品牌站套路，而是像纪录片章节：先建立环境，再解释机制，再给出局部证据和叙事收束。

01 / scene

强视觉起手

用一整块深色海洋场景和高对比大标题建立世界观。

02 / system

内容像简报

用标题、说明、局部数据和图像一起维持科研可信度。

03 / flow

滚动像章节

每段只负责一个功能，不让页面沦为信息大杂烩。

内容页系统要从首页就埋下秩序，而不是滚到第二屏才开始补救。

模板把媒体区、正文区和引语区拆成三种节奏。这样你以后往里换真实摄影、地图、实验图像或人物特写时，结构不会散。

数据块、术语卡、图注和 CTA 都应该是同一套视觉语言的延续，而不是突然出现一排 SaaS 组件。

“高级科普网页不是把知识塞进网页，而是让知识拥有自己的舞台调度。” editorial note / template voice

适合作为研究项目首页、展览专题页、科学栏目头图页的起始模板。

把真实内容替换进去以后，它应该仍然像一页“被设计过的叙事”，而不是几个 section 的拼接。
