# 中文 Prompt

你现在要设计的不是“传统网课 UI”，而是一种“科学纪录片 + 数字展览 + 任务控制台”式的高级科普界面。整体审美必须摆脱培训机构感、SaaS 卡片感和儿童科普感，转而呈现叙事性、工业美学、信息精度和电影级视觉张力。

先选 1 个主气质，再选 1 个辅气质，不能四种平均混搭：

- OceanX：深海、沉浸、暗场、高对比、大字、视差、电影感
- Brooklyn Museum：博物馆、留白、黑白中性色、出版物感、严谨网格、静态高级
- NASA JPL：任务控制台、科学报告、模块秩序、信号色点缀、技术可信度
- MasterClass：高质量人物或主题摄影、流媒体式内容呈现、非对称拼贴、海报感

设计目标：

1. 首页像海报，而不是课程目录。
2. 内容页像数字图录或科学任务档案，而不是普通文章模板。
3. 图像、正文、术语、图注、数据和 CTA 属于同一套视觉宇宙。
4. 既有电影感，也有编辑感和工程感。

首页规则：

- 首屏只保留一个主视觉、一组标题和一个主 CTA。
- 用全幅媒体、非对称栅格和滚动节奏建立叙事。
- 每个 section 只承担一个任务。
- 不要一开始就堆卡片、徽章、tab 和小组件。

内容页规则：

- 开头必须包含分类、标题、日期、阅读时长、作者或任务信息。
- 正文列宽稳定克制，适合长时间阅读。
- 长文要拆成章节标题、图文块、图注、引语、数据块、术语解释、相关推荐。
- 图片、视频、地图、图示、3D 模型都要像正文叙事的一部分。
- 图注要像展签或科研图注，短、准、安静。

排版规则：

- 标题有展示感，正文有阅读稳定性。
- 字号级差明确，不要所有字都差不多大。
- 中文技术内容统一一个中文字体家族；拉丁字母、数字和公式统一一个配套家族。
- 所有变量、上下标、Greek letters 和公式片段必须走统一渲染规则。

严格避免：

- 网课平台感
- SaaS 卡片墙
- 模板化暗色科技风
- 廉价发光描边
- 首页高级、内页普通
- 装饰性强但没有叙事价值的媒体

最终输出时，请确保：

- 第一眼建立世界观
- 长内容仍然高级且可读
- 桌面、平板、手机三档都稳定
- 页面整体更像“数字展览 + 科学纪录片 + 精密内容系统”，而不是“在线课程平台”
